<a name="about"></a>
## 📌 Sobre el proyecto

**Boquerones a la calle** es una aplicación web interactiva que actúa como un guía local de Málaga propulsado por IA. Permite a los usuarios descubrir planes y eventos (conciertos, fútbol, cultura) charlando de manera natural con "El Boquerón", un asistente virtual con acento malagueño.

---

<a name="requirements"></a>
## 🧰 Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge).
- Servidor web local para desarrollo.

---

<a name="installation"></a>
## ⚙️ Instalación

1. Obtén los archivos del proyecto (si no lo has hecho ya).
2. Abre la carpeta del proyecto en tu editor de código preferido.

---

<a name="usage"></a>
## ▶️ Cómo ejecutar

Utiliza los scripts de inicio incluidos en la raíz del proyecto:

**Windows:**
```cmd
start.cmd
```

**macOS / Linux:**
```bash
./start.sh
```

Luego abre tu navegador en la dirección indicada por el servidor local (ej. `http://localhost:8080` o similar).

---

<a name="stop"></a>
## ⏹ Cómo parar

**Windows:**
```cmd
stop.cmd
```

**macOS / Linux:**
```bash
./stop.sh
```
